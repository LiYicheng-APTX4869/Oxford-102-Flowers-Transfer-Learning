# 基于迁移学习的 Oxford 102 Flowers 花卉分类

本项目基于 **PyTorch** 在 **Oxford 102 Category Flower Dataset** 上完成花卉图像分类实验，重点关注以下问题：

- 基于 **ImageNet 预训练卷积神经网络** 的微调效果
- 学习率和训练轮数等超参数对性能的影响
- 预训练与随机初始化之间的性能差异
- 注意力模块与轻量级 Vision Transformer 在该任务上的表现

该仓库对应课程实验项目中的“任务1：微调在 ImageNet 上预训练的卷积神经网络实现花卉识别”。

---

## 一、项目简介

本实验以 **ResNet-18** 为主干网络，围绕 Oxford 102 Flowers 数据集构建了完整的图像分类实验流程，包括：

- 基于 ImageNet 预训练参数的 ResNet-18 微调 baseline
- 学习率对比实验
- 训练轮数对比实验
- 预训练消融实验：`pretrained` vs `random init`
- 注意力机制实验：`SE`、`CBAM`
- 轻量级 Transformer 实验：`ViT-Tiny`

实验重点不仅在于获得较高的分类精度，也在于系统分析不同训练策略与不同模型结构在小样本花卉分类任务中的实际差异。

---

## 二、数据集简介

实验使用 **Oxford 102 Category Flower Dataset**，该数据集包含：

- 102 个花卉类别
- 8189 张图像

本项目采用官方标准划分：

- `train = 1020`
- `val = 1020`
- `test = 6149`

由于训练集规模相对较小，因此该任务具有明显的小样本特征，也非常适合研究迁移学习在细粒度图像分类中的效果。

---

## 三、方法概述

### 1. Baseline

主模型采用 **ResNet-18**。整体做法如下：

- backbone 使用 ImageNet 预训练参数初始化
- 最后一层全连接替换为 102 类输出
- 分类头从头训练
- backbone 与分类头采用分层学习率

这种设置兼顾了预训练特征保留与目标任务适配能力，是本实验中最稳定有效的训练策略。

### 2. 超参数分析

围绕 baseline，主要分析了以下两类超参数：

- 学习率组合
- 训练轮数（epoch）

通过受控对比，观察不同配置对验证集和测试集性能的影响。

### 3. 预训练消融实验

为了验证迁移学习的价值，实验将相同结构的 ResNet-18 分别在以下两种条件下进行训练：

- 使用 ImageNet 预训练权重初始化
- 使用随机初始化从头训练

同时，为避免“随机初始化模型仅仅是训练不足”的误判，又补做了更长训练轮数的随机初始化实验。

### 4. 结构扩展实验

在 baseline 基础上，进一步测试了以下扩展模型：

- `ResNet-18 + SE`
- `ResNet-18 + CBAM`
- `ViT-Tiny (scratch)`

这些实验用于比较传统 CNN、注意力增强 CNN 与轻量级 Transformer 在 Flowers-102 小样本场景下的实际表现差异。

## 四、实验结果总览

主要实验结果如下：

| 实验 | Best Val Acc | Test Acc | Test Macro-F1 | 备注 |
|---|---:|---:|---:|---|
| ResNet-18，预训练，30 epoch | 0.9392 | 0.9205 | 0.9127 | baseline |
| ResNet-18，预训练，40 epoch | **0.9392** | **0.9257** | **0.9205** | 最优结果 |
| ResNet-18，随机初始化，30 epoch | 0.4775 | 0.4124 | 0.3831 | 训练明显不足 |
| ResNet-18，随机初始化，60 epoch | 0.5627 | 0.5048 | 0.4860 | 有提升，但仍明显落后 |
| ResNet-18 + SE | 0.9196 | 0.9089 | 0.8999 | 未超过 baseline |
| ResNet-18 + CBAM | 0.9078 | 0.8769 | 0.8691 | 低于 baseline |
| ViT-Tiny（scratch），40 epoch | 0.3284 | 0.2747 | 0.2253 | 表现较弱 |
| ViT-Tiny（scratch），80 epoch | 0.4412 | 0.3828 | 0.3547 | 有提升，但仍不理想 |

## 五、主要实验结论

### 1. 预训练是 Flowers-102 上最关键的因素

使用 ImageNet 预训练参数初始化的 ResNet-18 明显优于随机初始化模型。即使将随机初始化模型训练到 60 epoch，其测试准确率也仅为 `0.5048`，而预训练 ResNet-18 的最佳测试准确率达到 `0.9257`。这说明在 Flowers-102 这类小样本细粒度分类任务中，迁移学习具有决定性作用。

### 2. 学习率设置会显著影响微调效果

三组学习率实验表明，过小的学习率会导致模型收敛明显变慢，最终性能也更差。综合稳定性和最终测试表现来看，`backbone_lr=1e-4`、`head_lr=1e-3` 是本项目中最稳妥的微调配置。

### 3. 增加训练轮数带来小幅但稳定的收益

在预训练 ResNet-18 上，将训练轮数从 30 增加到 40 后，测试准确率由 `0.9205` 提升到 `0.9257`。虽然提升幅度不算很大，但结果稳定，说明在当前数据规模下，适度延长训练仍然能够带来收益。

### 4. 注意力模块没有带来性能提升

SE 和 CBAM 两种注意力模块在本实验中的表现均未超过原始 ResNet-18 baseline。其中：

- `SE` 相对更稳定
- `CBAM` 效果更弱

这说明注意力机制的收益并不是绝对的，在中小规模数据集与固定训练设置下，额外结构复杂度并未转化为更高精度。

### 5. ViT-Tiny 在该任务上不占优势

ViT-Tiny 从头训练 40 epoch 时表现较差，即使训练到 80 epoch 后，测试准确率也只有 `0.3828`。这与 Transformer 架构对大规模数据或强预训练依赖较高的经验现象一致，也说明在 Flowers-102 这类任务上，预训练 CNN 仍然更加实用。

## 六、最终模型

本项目最终采用的最优模型为：

- **ResNet-18**
- **ImageNet 预训练初始化**
- **40 epoch 微调**

其测试集结果为：

- **Test Accuracy = 92.57\%**
- **Test Macro-F1 = 0.9205**

这一结果也是整个项目中所有实验设置下的最佳结果。

## 八、结果概括


> 在 Oxford 102 Flowers 这一小样本花卉分类任务上，**基于 ImageNet 预训练的 ResNet-18 微调方案是最稳定、最有效的选择**。

从实验结果来看，在数据规模有限的情况下，预训练带来的收益明显强于简单增加模型复杂度。这也是本项目最核心的实验结论之一。
